# Commander Scout Review Issue Archive

This directory contains the older architecture and design issue writeups that were generated from the November 2025 review cycle.

## Status

**Last Updated:** 2026-03-31  
**Purpose:** Historical planning archive

These markdown issue files are still useful as backlog context, but they are no longer a reliable source of present-tense implementation status. Several items from that original review set have already landed in the codebase, including:

- IndexedDB caching
- request deduplication
- virtualized table rendering
- route and component lazy loading
- Dependabot configuration
- Content Security Policy and CSP reporting

## How To Use This Folder

- Use these files for historical rationale and previously discussed implementation ideas.
- Do not assume an issue is still open just because a markdown file exists here.
- Check the current review set in [../REVIEWS_COMPLETE.md](../REVIEWS_COMPLETE.md) before recreating backlog items.
